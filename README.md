2.openclash 配置好以后，进行备份，以后如有问题，能进行恢复吗，方法是什么？
  OpenClash 配置备份后可以恢复，有三种可靠方法，按推荐优先级排序如下，适配不同场景与故障类型。
  一、图形化快速恢复（推荐，适合新手）
  1. 先完成备份（版本更新页）
  LuCI → 服务 → OpenClash → 版本更新 → 备份设置，选择要备份的内容（配置 / 规则 / 内核），点击 “备份”，下载生成的 .tar.gz 压缩包并妥善保存。
  2. 恢复步骤（配置文件管理页）
  进入 OpenClash → 配置文件管理 → 备份与恢复，点击 “上传备份文件”，选择之前保存的 .tar.gz 压缩包。
  上传完成后点击 “恢复”，等待提示成功后，重启 OpenClash 服务：/etc/init.d/openclash restart。
  验证：检查代理、分流规则是否恢复正常，日志无报错即可。
